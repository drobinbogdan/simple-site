Добро пожаловать на мою веб-страницу

Эта страница создана в рамках индивидуального проекта по изучению Git и GitHub.

Цель работы — продемонстрировать базовые навыки веб-разработки и использование системы контроля версий.

О проекте

В проекте представлены два файла — index.html и style.css, которые хранятся в репозитории GitHub.

Страница оформлена с помощью простых CSS-стилей, задающих цвет фона, выравнивание текста и внешний вид заголовков.
